### API ENDPOINT

- `POST /register` = create new User
- `POST /login` = login the apllication
- `POST /google-sign` = login the apllication with gmail

- `GET /tour` = show all data Tour
- `GET /tour/:id` = show all data Tour by Id
- `GET /spesifictour`

### POST /register

#### Request

- Body

```json
{
  "username": String,
  "email": String,
  "password": String,
  "phoneNumber": String,
  "address": String
}

```

#### Response

_201_

```json
{
  "id": Integer,
  "email": String
}


```
